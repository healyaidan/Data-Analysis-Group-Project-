# Group 15 — Project 15: Measles susceptibility in Glasgow (Primary school children)

## Overview
This repository contains the collaborative R/Quarto workflow for the **Data Analysis Group Project (Group 15)**.

**Project 15** uses Scottish Childhood Immunisation Record System (SCIRS)-derived data on **measles susceptibility** among **primary school children** across **133 Intermediate Zones (IZs) in Glasgow**, covering **1998–2012**.

Deliverable: a concise **Quarto report** (max **6 pages**) summarising:
- Introduction (dataset + question of interest)
- Exploratory analysis
- Formal statistical analysis
- Conclusions
